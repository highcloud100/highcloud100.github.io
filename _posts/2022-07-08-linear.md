<h3 id="linear-equations">linear equations</h3>
<hr>
<ul>
<li>m = equations</li>
<li>n = unkowns</li>
</ul>
<ol>
<li>m &gt; n
<ul>
<li>over-determined : 부정식</li>
<li>under complete</li>
<li>주로 Lidge를 사용한다함</li>
<li>거의 대부분의 머신러닝 문제</li>
</ul>
</li>
<li>m = n
<ul>
<li>흔히 학교에서 풀었던 방정식</li>
<li>unique</li>
</ul>
</li>
<li>m &lt; n
<ul>
<li>over complete</li>
<li>해가 무수히 존재함</li>
<li>under-determined</li>
<li>주로 Lasso를 사용한다함</li>
</ul>
</li>
</ol>
